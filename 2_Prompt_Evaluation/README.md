# Prompt Evaluation

Techniques for systematically evaluating and improving prompts using the Claude API.

## Prerequisites

- Python 3.10+
- `anthropic` and `python-dotenv` packages
- An Anthropic API key in a `.env` file (`ANTHROPIC_API_KEY=sk-...`)

## Notebooks

### 1. Prompt Eval Workflow

Set up a prompt evaluation workflow to systematically test and compare prompt variations. Covers the foundational patterns for measuring prompt quality and iterating on prompt design.

**Key concepts:** prompt evaluation, test cases, prompt comparison, systematic prompt iteration
